# Tactical RPG — POC

Dungeon crawler tactique au tour par tour, style Wakfu/Dofus.

## Lancer le projet

```bash
npm install
npm run dev
```

Ouvre [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Stack

- React + TypeScript
- Phaser 3 (moteur de jeu)
- Vite
